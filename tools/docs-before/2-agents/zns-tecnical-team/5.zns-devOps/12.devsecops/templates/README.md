# 📋 Templates FinDevSecOps

Este directorio contiene plantillas reutilizables para el agente Senior FinDevSecOps Engineer.

## 📁 Estructura

```
templates/
├── README.md                           # Este archivo
├── security/
│   ├── security-assessment.md          # Template de evaluación de seguridad
│   ├── incident-response-playbook.md   # Playbook de respuesta a incidentes
│   └── compliance-checklist.md         # Checklist de compliance
├── finops/
│   ├── cost-analysis-report.md         # Template de análisis de costos
│   ├── monthly-report.md               # Reporte mensual FinOps
│   └── optimization-proposal.md        # Propuesta de optimización
├── pipeline/
│   ├── github-actions-devsecops.yml    # Pipeline GitHub Actions
│   └── gitlab-ci-devsecops.yml         # Pipeline GitLab CI
├── policies/
│   ├── opa-policies/                   # Políticas OPA/Rego
│   └── kyverno-policies/               # Políticas Kyverno
└── runbooks/
    └── incident-runbook-template.md    # Template de runbook
```

## 🚀 Uso

1. Copia la plantilla necesaria
2. Personaliza según el contexto del proyecto
3. Valida con el checklist correspondiente
